# React Internship Project

Welcome to the React Internship Project repository! This project serves as a demonstration of my skills in React development during my internship.


## Installation

To run this project locally, follow these steps:

1. Clone the repository to your local machine using the following command:
   
   **git clone https://github.com/harshdev4/react-internship-project.git**


3. Navigate to the project directory:
   
   **cd react-internship-project**

   
5. Install dependencies using npm or yarn:
   
   **npm install**


7. Start the development server:
   
   **npm run dev**
